# Café Rush — Bar Mode

High-speed barista practice: read the ticket, build the drink, and serve before the clock expires.

## Run locally

```bash
python -m http.server
```

Open `http://localhost:8000/apps/cafe-rush/`.

## Gameplay notes
- Start your shift to spawn random hot and iced drink tickets.
- Select a cup first to reset the build area, then stack ingredients in any order.
- Press **Serve Order** to check your build, earn points, and move to the next ticket.
- At the end of the 60-second shift, download a quick performance report.
