# Mail Workflow Simulator

Mail Workflow Simulator combines the quick inbox triage flow from Outlook Simulation with the coached reply drafting flow from Inbox Simulator.

## What it includes

- Outlook-style inbox review with `Reply`, `Archive`, and `Delete` actions
- Mixed message set with quick triage tasks, phishing traps, and coached reply scenarios
- Draft composer with live heuristics for greeting, sign-off, spelling, and scenario goals
- Session timer, accuracy tracking, reply quality scoring, downloadable report, and restart flow

## How to run

Serve the repository root with a static server, then open:

`/apps/mail-workflow-simulator/index.html`
